#  🚀 Tecnologias Utilizadas

-  TypeScript
-  Ethereal-Email
-  Express
-  TypeORM



# ⚙️ Como executar

-   Clone o repositório
```bash
    $ git clone 'repository'
```
-   Instale as dependências com yarn
```bash
    $ npm i
```
-   Inicie o servidor com yarn dev
```bash
    $ npm run
```
A aplicação pode ser acessada em localhost:3333.
